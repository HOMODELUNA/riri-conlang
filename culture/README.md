# Riri 文化

## 民族

水獭型兽人组成的部族/城市。

## 生活方式

渔猎是生活的大部分。

## 居住

住在船上，城市则是港湾。

## 文化观念

人 + 船视为一个整体，因此"停泊在上海"与"对船做出停在上海的行为"在语义上等价，可视为双宾语结构。

---

## 亲缘与家族关系（loop2）

由于长期居住在海上的船只，受船只空间限制，每个家庭单位较小。现实世界观念中的庞大家族难以形成，因此各家庭散居。但由于海上船只文明的规模限制，一个社区内的兽人通常出自一个或几个家族。

相关词汇：`luh`（船/居所）、`luuh`（家庭/家）、`voluh`（社区）、`lumar`（家族，合成词）。

## 进制（loop2）

水獭兽人不具备人类一样的五根手指，其进制参照六肢：头、手（左/右）、足（左/右）、尾。

因此 Riri 采用**六进制**，数字 `aw`(1) `lik`(2) `vik`(3) `fu`(4) `max`(5) `les`(6) 均与身体部位存在古义关联。
