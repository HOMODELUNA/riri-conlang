# Riri 语法

Riri 是一种为水獭型兽人部族设计的造语，语法以简单、快速会意为核心。

---

## 音系

### 辅音

| 双唇 | 齿龈 | 软腭 |
|------|------|------|
| p / b | t / d | k / g |
| m      | n      |      |
| f / v  | s / z  |      |
|        | sh / j / ch / dj / ts | |
|        | r / l  | h /x/ |
| w      | y      |      |

- `h` 表示 /x/，词内不使用 `x` 拼写 /x/。
- `ts` 作为塞擦音拼写使用。
- 不区分清送气音和清音，避免使用 `ph`、`th`、`kh` 表示送气对立。

### 元音

a, e, i, o, u

- 支持双元音
- 支持长元音

---

## 句法

| 特征           | 设定                |
|----------------|---------------------|
| 类型           | 分析语              |
| 形名关系       | OA（形容词→名词）   |
| 动宾关系       | VO（动词→宾语）     |
| 双宾语         | 支持（人+船视为整体）|

---

## 语法范畴

| 范畴 | 发达程度 | 说明                                   |
|------|----------|----------------------------------------|
| 性   | 不区分   | —                                      |
| 数   | 不区分   | 无单复数，有量词                       |
| 格   | 不区分   | 依赖语序和双宾语                       |
| 时   | 不区分   | 依赖副词                               |
| 体   | 简单区分 | 未完成体 / 完成体 / 成就体，使用副词标记 |

---

## 修饰词顺序（loop2）

修饰词遵循以下语序规则：

- **名词短语**：`[数词] [名词] [性质] [状态]`
- **时间词**：永远放在句首或句尾，以突出时间信息
- **动词短语**：`[频率] [动作] [副词]`

海上环境变化多端，数量/频率信息优先级较高。

---

## 判断词（loop2）

不存在独立的判断词"是"。名词直接相连即表判断关系。

- 例：`A nua`（我是学生）

---

## 体标记（loop3）

Riri 使用副词标记体的差异，根据事件是否达成所需的效果，区分完成体和成就体。

### 未完成体

`me` / `e`，表示正在进行。

- 前词以元音结尾时，用 `me`
- 前词以辅音结尾时，用 `e`

> 例：`A me nulir biri`（我正在装鱼）

### 完成体

`dun`，置于动词前。

> 例：`Nayum rai, amra dun rahamrei`（今天是满月，我们去赶海了）

### 成就体

`daa`，置于动词后。

> 例：`Ye fuha daa mrai vik`（他走过三片大海）
> 例：`Huyum a dissi zir daa`（昨天我没钓到鱼）

---

## 词性混同

作为海上渔猎种族，Riri 以快速会意为重点，词性之间可以相对省略或混同。
