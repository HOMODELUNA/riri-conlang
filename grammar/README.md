# Riri 语法

Riri 是一种为水獭型兽人部族设计的造语，语法以简单、快速会意为核心。

---

## 音系

### 辅音

| 双唇 | 齿龈 | 软腭 |
|------|------|------|
| p / b | t / d | k / g |
| m      | n      |      |
| f / v  | s / z  |      |
|        | sh / j / ch / dj | |
|        | r / l  | h (x) |
| w      | y      |      |

### 元音

a, e, i, o, u

- 支持双元音
- 支持长元音

---

## 句法

| 特征           | 设定                |
|----------------|---------------------|
| 类型           | 分析语              |
| 形名关系       | OA（形容词→名词）   |
| 动宾关系       | VO（动词→宾语）     |
| 双宾语         | 支持（人+船视为整体）|

---

## 语法范畴

| 范畴 | 发达程度 | 说明                                   |
|------|----------|----------------------------------------|
| 性   | 不区分   | —                                      |
| 数   | 不区分   | 无单复数，有量词                       |
| 格   | 不区分   | 依赖语序和双宾语                       |
| 时   | 不区分   | 依赖副词                               |
| 体   | 简单区分 | 未完成体 / 完成体，使用副词标记         |

---

## 词性混同

作为海上渔猎种族，Riri 以快速会意为重点，词性之间可以相对省略或混同。
