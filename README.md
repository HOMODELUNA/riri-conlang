# Riri

水獭型兽人部族的造语（conlang），语法以简单、快速会意为核心。

---

## 螺旋驱动的造语

Riri 的构建采用**螺旋驱动**的工作流：每次讨论产生一个 **loop**，包含语法、文化、词语、例句的设计，然后对应更新各项目文件。

### 流程

```
讨论 → loop（语法/文化/词语/例句） → 更新 grammar/ culture/ riri.xml 等
```

每个 loop 对应一次设计迭代，逐步完善语言体系。

### 已完成的 Loops

| Loop | 内容               | 文件              |
|------|--------------------|-------------------|
| 1    | 初始螺旋：音系、语法、词汇、文化 | `loops/loop1.md` |

### 更新规则

- **语法变更** → 更新 `grammar/README.md`
- **文化变更** → 更新 `culture/README.md`
- **词汇变更** → 更新 `riri.xml`（`odict compile` 验证）
- **例句变更** → 更新 `riri.xml` 例句 + `grammar/README.md`
